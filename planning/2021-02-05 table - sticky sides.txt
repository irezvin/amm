Sticky header and footer, overlay

-   Фиксированный верх с position: sticky

    Фиксированный заголовок делается очень просто:
    https://www.w3docs.com/snippets/html/how-to-create-a-table-with-a-fixed-header-and-scrollable-body.html

    -   вся таблица в div
    -   у div ограничена высота, scroll

-   Фиксирование любых элементов при помощи краев (используется в handsontable)

    i   Div'ы с клонами размещаются поверх таблицы.
        Их можно скрывать, когда прокрутка доходит до конца (или оставлять, без разницы).

        Для клонирования мне нужно клонировать представления, _в том числе_ представления DisplayParent'ов 
        с подчиненными Visual. То есть DisplayParent должен поддерживать клонирование.

    -   клонирование представлений
    -   моя боль: у элемента может быть разное количество displayParent'ов
    -   клонирование представлений (а не элементов) достаточно простое
